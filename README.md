# **Twitter toolbox**
Este respositório contém uma série de ferramentas úteis (e relativamente simples para iniciantes) para se trabalhar com dados do Twitter.
No trabalho desenvolvido, essas ferramentas foram aplicadas utilizando a API v2 do Twitter e a biblioteca tweepy para a mineração dos dados, e a linguagem Python, juntamente com algumas outras bibliotecas, para limpeza e análise.
> OBS.: Todo o material utilizado está referenciado nesse README. Qualquer dúvida é só procurar a seção "Referências"
